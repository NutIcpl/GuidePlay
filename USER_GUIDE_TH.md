## 1. เริ่มต้นใช้งาน - การเข้าสู่ระบบ

### 1.1 หน้าแรก (Landing Page)

เมื่อเปิดเว็บไซต์ คุณจะพบหน้าแรกที่มีตัวเลือก:

- **"Get Started Free"** หรือ **"Sign In"** - สำหรับผู้ที่ต้องการสร้างและจัดการ Quiz (Host)
- **"Join Game"** - สำหรับผู้เล่นที่ต้องการเข้าร่วมเกม

### 1.2 การเข้าสู่ระบบ (Login)

**เส้นทาง:** `/auth/login`

มี 2 วิธีในการเข้าสู่ระบบ:

#### วิธีที่ 1: ใช้บัญชี Microsoft (แนะนำ)

1. คลิกปุ่ม **"Sign in with Microsoft"**
2. ระบบจะเปิดหน้าต่าง Microsoft OAuth
3. เข้าสู่ระบบด้วยบัญชี Microsoft ของคุณ
4. อนุญาตให้แอปเข้าถึงข้อมูล
5. ระบบจะนำคุณไปยังหน้า Dashboard โดยอัตโนมัติ

#### วิธีที่ 2: สมัครสมาชิกใหม่ด้วย Email

**เส้นทาง:** `/auth/register`

1. คลิกลิงก์ **"Register"** หรือ **"สมัครสมาชิก"**
2. กรอกข้อมูล:
   - **Full Name:** ชื่อ-นามสกุล ของคุณ
   - **Email:** อีเมลที่ใช้งานได้จริง
   - **Password:** รหัสผ่าน (อย่างน้อย 6 ตัวอักษร)
   - **Confirm Password:** ยืนยันรหัสผ่าน
3. คลิกปุ่ม **"Create Account"**
4. ระบบจะสร้างบัญชีและนำคุณไปยังหน้า Dashboard

> **หมายเหตุ:** หากมีบัญชีอยู่แล้ว ให้คลิก "Already have an account? Sign in" เพื่อกลับไปหน้า Login

## 2. หน้า Dashboard - ศูนย์กลางการจัดการ

**เส้นทาง:** `/host/dashboard`

หลังเข้าสู่ระบบสำเร็จ คุณจะมาถึงหน้า Dashboard ซึ่งเป็นศูนย์กลางการจัดการทุกอย่าง

### 2.1 ส่วนประกอบของหน้า Dashboard

#### แถบนำทางด้านซ้าย (Sidebar)

| ไอคอน | เมนู | คำอธิบาย |
|-------|------|----------|
| 🏠 | **My Quizzes** | ดู Quiz ทั้งหมดที่คุณสร้าง |
| ➕ | **Create New** | สร้าง Quiz ใหม่ |
| 📊 | **Analytics** | ดูสถิติและผลการเล่น |
| ❓ | **How to** | คำแนะนำการใช้งาน |
| 🏢 | **Enterprise** | ฟีเจอร์สำหรับองค์กร |

#### พื้นที่แสดง Quiz

- **มุมมองแบบ Grid:** แสดง Quiz เป็นการ์ดภาพ
- **มุมมองแบบ List:** แสดง Quiz เป็นรายการ

### 2.2 การจัดการ Quiz ที่มีอยู่

แต่ละ Quiz Card จะแสดง:
- **ชื่อ Quiz**
- **จำนวนคำถาม**
- **Theme ที่ใช้**

**ปุ่มควบคุมบน Quiz Card:**

| ปุ่ม | การทำงาน |
|------|----------|
| ▶️ **Play** | เริ่มเกมใหม่กับ Quiz นี้ |
| ✏️ **Edit** | แก้ไข Quiz |
| 📋 **Duplicate** | สำเนา Quiz |
| 🗑️ **Delete** | ลบ Quiz (ต้องยืนยัน) |

## 3. การสร้าง Quiz ใหม่

**เส้นทาง:** `/host/dashboard/create`

### 3.1 ขั้นตอนการสร้าง Quiz

#### ขั้นตอนที่ 1: ตั้งค่าพื้นฐาน

1. คลิก **"Create New"** จาก Sidebar หรือ Dashboard
2. กรอกข้อมูลพื้นฐาน:

| ฟิลด์ | คำอธิบาย | ตัวอย่าง |
|-------|----------|----------|
| **Quiz Name** | ชื่อของ Quiz | "ทดสอบความรู้ทั่วไป" |
| **Description** | คำอธิบายสั้นๆ | "แบบทดสอบสำหรับทีมขาย Q4" |

#### ขั้นตอนที่ 2: เลือก Theme

คลิกที่ **Theme Picker** เพื่อเลือกธีมสีสำหรับ Quiz:

- **Classic** - สีน้ำเงินคลาสสิก
- **Neon** - สีนีออนสดใส
- **Dark** - โทนสีเข้ม
- **Nature** - โทนสีธรรมชาติ
- **Ocean** - โทนสีทะเล
- และอื่นๆ อีกมากมาย

#### ขั้นตอนที่ 3: ตั้งค่าเกม (Settings)

| การตั้งค่า | คำอธิบาย | ค่าที่เลือกได้ |
|------------|----------|---------------|
| **Auto-advance Time** | เวลาหน่วงก่อนเปลี่ยนคำถามถัดไป | 3-30 วินาที |
| **Public/Private** | Quiz สาธารณะหรือส่วนตัว | เปิด/ปิด |
| **Team Mode** | เปิดโหมดทีม | เปิด/ปิด |
| **Max Teams** | จำนวนทีมสูงสุด (ถ้าเปิด Team Mode) | 2-4 ทีม |
| **Auto-read Questions** | อ่านคำถามอัตโนมัติ (TTS) | เปิด/ปิด |
| **Self-paced Mode** | อนุญาตให้เล่นคนเดียว | เปิด/ปิด |
| **Passing Score** | คะแนนผ่าน (สำหรับ Self-paced) | 0-100% |

### 3.2 การเพิ่มคำถาม

#### การเพิ่มคำถามด้วยตนเอง

1. คลิกปุ่ม **"+ Add Question"**
2. กรอกข้อมูลคำถาม:

| ฟิลด์ | คำอธิบาย |
|-------|----------|
| **Question Text** | ข้อความคำถาม |
| **Question Image** | รูปภาพประกอบ (ไม่บังคับ) |
| **Time Limit** | เวลาในการตอบ (5-60 วินาที) |
| **Points** | คะแนนของคำถาม (100-5000) |

3. เพิ่มตัวเลือกคำตอบ (2-4 ตัวเลือก):
   - กรอกข้อความในแต่ละช่อง
   - คลิกเลือกคำตอบที่ถูกต้อง (จะมีเครื่องหมาย ✓)

4. คลิก **"Save Question"** หรือกด Enter

#### การจัดการคำถาม

| ปุ่ม | การทำงาน |
|------|----------|
| ⬆️ | เลื่อนคำถามขึ้น |
| ⬇️ | เลื่อนคำถามลง |
| ✏️ | แก้ไขคำถาม |
| 🗑️ | ลบคำถาม |

### 3.3 การบันทึก Quiz

1. ตรวจสอบคำถามทั้งหมดว่าถูกต้อง
2. คลิกปุ่ม **"Save Quiz"** ที่ด้านบนหรือด้านล่าง
3. ระบบจะบันทึกและแจ้งเตือนเมื่อสำเร็จ
4. Quiz จะปรากฏในหน้า Dashboard

## 4. การใช้ AI สร้างคำถามอัตโนมัติ

### 4.1 เปิดใช้งาน AI Quiz Generator

1. ในหน้าสร้าง Quiz คลิกปุ่ม **"Generate with AI"** หรือ **"AI Generator"**
2. หน้าต่าง AI Generator จะเปิดขึ้น

### 4.2 การกำหนดค่าสำหรับ AI

| ฟิลด์ | คำอธิบาย | ตัวอย่าง |
|-------|----------|----------|
| **Topic** | หัวข้อที่ต้องการสร้างคำถาม | "ประวัติศาสตร์ไทยสมัยรัตนโกสินทร์" |
| **Number of Questions** | จำนวนคำถามที่ต้องการ | 10 |
| **Difficulty** | ระดับความยาก | Easy / Medium / Hard |
| **Language** | ภาษาของคำถาม | Thai / English |

### 4.3 ขั้นตอนการสร้าง

1. กรอกข้อมูลตามต้องการ
2. คลิก **"Generate"**
3. รอระบบ AI ประมวลผล (ประมาณ 10-30 วินาที)
4. ตรวจสอบคำถามที่สร้างขึ้น
5. คลิก **"Add to Quiz"** เพื่อเพิ่มคำถามทั้งหมด
6. หรือเลือกเฉพาะคำถามที่ต้องการ

> **เคล็ดลับ:** คุณสามารถแก้ไขคำถามที่ AI สร้างได้หลังจากเพิ่มเข้า Quiz แล้ว

## 5. การเริ่มเกมและจัดการ Lobby

### 5.1 การเริ่มเกมใหม่

1. ไปที่หน้า Dashboard
2. หา Quiz ที่ต้องการเล่น
3. คลิกปุ่ม **▶️ Play** บน Quiz Card
4. ระบบจะสร้างเกมใหม่และเปิดหน้าต่างใหม่

### 5.2 หน้า Host Lobby

```
│                                             │

│                                             │
└─────────────────────────────────────────────┘
```

#### องค์ประกอบสำคัญ

| ส่วน | คำอธิบาย |
|------|----------|
| **QR Code** | ผู้เล่นสแกนเพื่อเข้าเกม (คลิกเพื่อขยาย) |
| **Game PIN** | รหัสเกม 6 ตัวอักษร |
| **URL** | ลิงก์สำหรับเข้าเกม |
| **Players Joined** | จำนวนผู้เล่นที่เข้าร่วม |
| **Player List** | รายชื่อและ Avatar ของผู้เล่น |
| **Start Game** | ปุ่มเริ่มเกม |

### 5.3 เสียงและดนตรี

- **เสียงเข้าร่วม:** ดังเมื่อมีผู้เล่นใหม่เข้าร่วม
- **เพลงประกอบ:** เริ่มเล่นอัตโนมัติเมื่อมีการคลิกบนหน้าจอ
- **ปุ่มควบคุมเสียง:** มุมขวาบนสำหรับเปิด/ปิดเสียง

### 5.4 การเริ่มเกม

1. รอให้ผู้เล่นเข้าร่วมครบ
2. ตรวจสอบรายชื่อผู้เล่น
3. คลิกปุ่ม **"Start Game"**
4. เกมจะเริ่มและทุกคนจะเห็นคำถามแรก

## 7. สำหรับผู้เล่น - การเข้าร่วมเกม

### 7.1 วิธีเข้าร่วมเกม

#### วิธีที่ 1: สแกน QR Code
1. เปิดกล้องมือถือ
2. สแกน QR Code ที่แสดงบนหน้าจอ Host
3. เปิดลิงก์ที่ปรากฏ

#### วิธีที่ 2: พิมพ์ URL
1. เปิด Browser
2. พิมพ์ URL: `yoursite.com/game/[GAME_PIN]`
3. แทนที่ `[GAME_PIN]` ด้วยรหัสเกม

#### วิธีที่ 3: จากหน้าแรก
1. ไปที่หน้าแรกของเว็บ
2. คลิก **"Join Game"**
3. ใส่ Game PIN

#### ขั้นตอนการลงทะเบียน

4. **เข้าร่วม:**
   - คลิกปุ่ม **"Join Game"**
   - รอ Host เริ่มเกม

- ชื่อและ Avatar ของคุณ
- ปุ่ม Emoji Reactions สำหรับส่ง reaction


#### วิธีตอบคำถาม

1. **อ่านคำถาม** ที่แสดงด้านบน
2. **รอตัวเลือกปรากฏ** (หน่วง 3 วินาทีเพื่ออ่านคำถาม)
3. **เลือกคำตอบ** โดยคลิกที่ตัวเลือก
4. **รอดูผลลัพธ์:**
   - ✅ **Correct!** - ตอบถูก (ได้คะแนน)
   - ❌ **Wrong** - ตอบผิด (แสดงคำตอบที่ถูก)
5. **ดูคะแนน** ที่ได้รับ
6. **รอคำถามถัดไป**

#### ระบบคะแนน

- **คะแนนพื้นฐาน:** ตามที่ Host กำหนด (100-5000 คะแนน)
- **Time Bonus:** ตอบเร็ว = คะแนนเพิ่มมากกว่า
- **คำนวณ:** คะแนน × (เวลาที่เหลือ / เวลาทั้งหมด)

## 8. โหมด Self-Paced (เล่นคนเดียว)

### 8.1 Self-Paced คืออะไร?

โหมดที่ผู้เล่นสามารถทำ Quiz ด้วยตัวเองได้ตลอดเวลา โดยไม่ต้องรอ Host เริ่มเกม เหมาะสำหรับ:
- การทบทวนบทเรียน
- การฝึกฝน
- การทดสอบตัวเอง
- การอบรมพนักงาน

### 8.2 การเปิดใช้งาน Self-Paced (สำหรับ Host)

1. ไปที่หน้าสร้าง/แก้ไข Quiz
2. เปิด **"Self-paced Mode"** ในส่วน Settings
3. กำหนด **Passing Score** (คะแนนผ่าน เช่น 70%)
4. บันทึก Quiz

### 8.3 การเล่นโหมด Self-Paced

**เส้นทาง:** `/play/[quizId]`

#### ขั้นตอน

1. **เข้าสู่หน้า Quiz:**
   - ได้รับลิงก์จาก Host
   - หรือ Host แชร์ลิงก์ `/play/[quizId]`

2. **หน้าลงทะเบียน:**

3. **ทำ Quiz:**
   - ตอบคำถามทีละข้อ
   - เห็นผลทันทีหลังตอบ (ถูก/ผิด)
   - คลิก "Next Question" เพื่อไปข้อถัดไป

### 8.4 ใบประกาศนียบัตร (Certificate)

หากผ่านเกณฑ์คะแนน:

1. คลิกปุ่ม **"Get Certificate"**

   - วันที่ทำ
4. สามารถดาวน์โหลดเป็นรูปภาพได้

## 10. คำถามที่พบบ่อย

### Q: ผู้เล่นสูงสุดต่อเกมมีกี่คน?
**A:** ไม่จำกัดจำนวน ขึ้นอยู่กับ capacity ของเซิร์ฟเวอร์

### Q: Quiz หนึ่งมีคำถามได้กี่ข้อ?
**A:** ไม่จำกัด แต่แนะนำ 10-30 ข้อเพื่อประสบการณ์ที่ดี

### Q: ถ้าผู้เล่น refresh หน้าจะเกิดอะไรขึ้น?
**A:** ระบบจะจำข้อมูลผู้เล่นและนำกลับเข้าเกมอัตโนมัติ

### Q: สามารถแก้ไข Quiz ระหว่างเกมได้ไหม?
**A:** ไม่ได้ ต้องจบเกมก่อนแล้วแก้ไขจาก Dashboard

### Q: Team Mode ทำงานอย่างไร?
**A:** ผู้เล่นเลือกทีม คะแนนจะรวมเป็นคะแนนทีม แสดง Team Leaderboard

### Q: ใช้บนมือถือได้ไหม?
**A:** ได้ ระบบรองรับ Responsive Design ทั้ง Desktop และ Mobile

### Q: เสียงไม่ดังเกิดจากอะไร?
**A:** Browser ต้องการให้ผู้ใช้คลิกบนหน้าก่อนเล่นเสียง ลองคลิกที่หน้าจอ

### Q: AI สร้างคำถามได้ภาษาไทยไหม?
**A:** ได้ เลือก Language เป็น Thai ในหน้า AI Generator
